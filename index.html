    • Error Time depends on level difficulty and upper limb impairment level, while assuming each subject reacts differently to harder mazes.
model_baseline <- lmer(Error_Time ~ Levels + FMA + (Levels|Subject),

Create Full Model:

  • Model description:

    • Error Time depends on maze difficulty x impairment interaction, with each subject having their own maze difficulty response.

      • Maze difficulty x impairment interaction: changes in upper limb impairment level effect how difficult a level is
model_full <- lmer(Error_Time ~ Levels*FMA + (Levels|Subject),
                   data = error_time_dataset_no5)
